# ◈ Eclyps Vision

> Transform any image into structured, reusable information.

An open-source image intelligence tool powered by Gemini. Upload any image and extract OCR text, detailed descriptions, semantic tags, accessibility alt text, and structured outputs with a single request.

---

## ✨ Features

* Image analysis
* OCR text extraction
* Detailed descriptions
* Semantic tags
* Accessibility alt text
* JSON export
* Markdown export
* CSV export
* Bring your own Gemini API key
* Open source
* No accounts
* No databases
* Lightweight

---

## Philosophy

**One image. One request. Everything organized.**

Eclyps Vision is built around simplicity. No accounts, no databases, no tracking, and no unnecessary complexity.

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/SombraPStudio/Eclyps---Vision.git
cd Eclyps---Vision
```

Launch a local server and open:

```
http://localhost:3000
```

---

## How It Works

1. Upload an image.
2. Enter your Gemini API key.
3. Analyze the image.
4. Receive:

* Description
* OCR text
* Semantic tags
* Alt text
* Structured outputs (JSON, Markdown, CSV)

---

## Privacy

Your API key is stored locally.

Images and API keys are never stored by Eclyps Vision. Requests are sent directly to Google's Gemini API.

Eclyps Vision does not use:

* Accounts
* Analytics
* Tracking
* Databases
* Ads
